北京周末出省游

Posted by cc-tian on 2019-04-27

鞍山

鞍山市位于辽宁省中南部，是国务院批准的具有地方立法权的较大的市，是中国境内重要的钢铁基地，有钢都之称。

3 星级酒店平均房价为 US$40，5 星级酒店平均房价为 US$47

保定

保定市位于河北省中部偏西。市境北临北京市、张家口市，东接廊坊市、沧州市，南邻石家庄市、衡水市，西界山西省大同市、忻州市。地处华北平原北部，太行山东麓，地势西高东低。

3 星级酒店平均房价为 US$30，5 星级酒店平均房价为 US$66

包头

包头市是内蒙古自治区下辖的最大城市，全国重要的冶金、材料和制造业基地，国务院批准设立具有地方立法权的较大的市，华北地区重要的工业城市。 包头市区由各具经济特色的3个中心街区组成。

3 星级酒店平均房价为 US$20，5 星级酒店平均房价为 US$51

日照

日照市位于山东省东南部。北与青岛市、潍坊市相连，西和临沂市、南与江苏省连云港市毗邻，东濒黄海。全市总面积5,359平方公里，人口超过300万。对外贸易和渔业是日照重要的经济组成部分，日照港是鲁南沿海的重要港口，其中24%的运货是煤炭，临港大工业的发展现在十分迅速。

3 星级酒店平均房价为 US$23

银川

银川市，简称银，古称中兴路、兴庆府、怀远镇、宁夏省城，是中华人民共和国宁夏回族自治区的首府，位于银川平原中部，西倚贺兰山，东临黄河。银川市为中国历史文化名城、中国·阿拉伯国家经贸论坛永久举办地，是全区军事、政治、经济、文化科研、交通和金融商业中心，历史上为西夏王朝首都。 银川市城市建成区面积为106平方千米。

3 星级酒店平均房价为 US$31，5 星级酒店平均房价为 US$84

巴彦淖尔市

巴彦淖尔市位于内蒙古西部，黄河北岸。市境东邻包头市，南接鄂尔多斯市，西界阿拉善盟，北临蒙古国。地处内蒙古高原西部，黄河自此折向东流，阴山山脉将市境分为两部，南麓为河套平原，北麓为乌拉特草原。全市总面积6.51万平方公里，人口167.73万。

3 星级酒店平均房价为 US$23。

阜新

阜新，是辽宁省辖地级市，位于辽宁省西部的低山丘陵区，是辽宁省西北部地区的中心城市，为沈阳经济区重要城市之一。1990年以来，该市地下可采煤大部分都被采空，经常发生地表塌陷等灾难，工人大量下岗，是资源枯竭城市的典型。

3 星级酒店平均房价为 US$20。

抚顺

抚顺市，清代及满洲国时期亦称抚西。1960年代文化大革命初期，曾非正式更名为雷锋市，位于辽宁省东部，是距离省会沈阳最近的城市，辖三县、四区，有两个省级开发区和一个省级高新技术产业开发区。是国务院批准的首批具有地方立法权的较大的市之一。中华人民共和国建国初期，曾被誉为“煤都”。

南海绿洲大酒店房价为 241人民币。

鄂尔多斯

鄂尔多斯市位于内蒙古自治区西南部，前身为伊克昭盟。市境东临呼和浩特市、山西省忻州市，南邻陕西省榆林市、宁夏吴忠市，西界阿拉善盟、乌海市、宁夏石嘴山市、银川市，北接包头市、巴彦淖尔市。地处鄂尔多斯高原，西北部为库布其沙漠，南部为毛乌素沙漠，东部为丘陵沟壑，北部为黄河冲积平原。

3 星级酒店平均房价为 US$32，5 星级酒店平均房价为 US$53

德州

德州市位于山东省西北部，黄河下游。东与济南、滨州两市相邻，南与聊城市相连，西接河北省衡水、邢台两市，北与河北省沧州市相接。市区邻近冀、鲁两省边界，京杭大运河流经境内。全市总面积10,358平方公里，人口574.23万。

3 星级酒店平均房价为 US$20

大同

大同市是山西省第二大城市，素有“魏都”“煤都”和“凤凰城”之称。

3 星级酒店平均房价为 US$32，5 星级酒店平均房价为 US$25

大连

大连市，旧称旅大市，别称鲲城、滨城，旧名达尔尼、青泥洼。位于中国辽东半岛辽宁省南端，东濒黄海，西临渤海。是中华人民共和国的5个计划单列市之一、15个副省级城市之一、也是全国14个沿海开放城市之一；也是东北亚国际航运中心、东北亚国际物流中心、区域性金融中心。

3 星级酒店平均房价为 US$40，5 星级酒店平均房价为 US$107

常州

常州市位于江苏省南部，大运河流经此地。长三角中心地带、衔太湖，与上海、南京、杭州皆等距相邻，并与苏州、无锡构成苏锡常都市圈。常州有文字记载的历史长达3200多年。 维基百科

3 星级酒店平均房价为 US$35，5 星级酒店平均房价为 US$74

沧州

沧州市位于河北省东部偏南。北界天津市、廊坊市，西北接保定市，西南连衡水市，东南邻山东省德州市、滨州市，东临渤海。地处华北平原东部，地势平坦。南运河、子牙新河流经境内。全市面积1.43万平方公里，人口744.3万。

盛泰国际酒店(5 星级酒店)US$53

赤峰

赤峰市原为昭乌达盟，位于内蒙古自治区东部，总面积8.69万平方公里，人口429.95万，是内蒙古自治区人口最多的地级市。市政府驻松山区。

3 星级酒店平均房价为 US$28，5 星级酒店平均房价为 US$80

阜阳

阜阳市位于安徽省西北部。市境东北界亳州市，东接淮南市，东南临六安市，西南邻河南省信阳市，西毗河南省驻马店市，西北河南省周口市。地处淮北平原南部，地势平坦。淮河流经南面边界，颍河自西北往东南贯穿并流经市区，于市境东南注入淮河。
